    <dl id="name-tips" class='tips'>
      <dt>Search Tips</dt>
      <dd>Enter a partial first name or leave the first name blank to expand your search</dd>
      <dd>For first name, try more formal variations (e.g., James rather than Jim or Jimmy) or vice versa</dd>
      <dd>If the last name is common, include the full first name and middle initial or city/state to narrow your results</dd>
      <dd>If the last name is hyphenated (e.g., Covey-Jones), try using one of the names, without the hyphen</dd>
      <dd>Make sure city name is spelled correctly or that the ZIP code is correct</dd>
      <dd>Try searching by name and state if you are unable to find a match</dd>
    </dl>

    <dl id="address-tips" class="tips" style="display: none">
      <dt>Search Tips</dt>
      <dd>Try searching by street name if you are not able to find the exact address</dd>
      <dd>Since street names can be abbreviated, try searching by house number, city and state to get broader results</dd>
      <dd>Try entering the entire city name (e.g., Mount Vernon rather than Mt. Vernon)</dd>
      <dd>Try dropping the "direction" in the name (e.g., for West Hanover, drop "West")</dd>
      <dd>If you are not finding results, try leaving the city blank and searching statewide</dd>
    </dl>

    <dl id="phone-tips" class="tips" style="display: none">
