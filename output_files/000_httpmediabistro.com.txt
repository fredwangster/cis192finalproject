<p><span style="font-weight:bold; text-decoration:none;"><a style="font-weight:bold; text-decoration:none;"
href="http://www.washingtonpost.com/lifestyle/style/pulitzer-prize-winners-announced/2012/04/16/gIQAsBF7LT_gallery.html#photo=1">Pulitzer Prize Winners Announced</a></span> (<i>Washington Post</i>)<br>
Fourteen winners of journalism's most prestigious annual prize were named by Columbia University's Graduate School of Journalism. <span style="font-weight:bold; text-decoration:none;"><a style="font-weight:bold; text-decoration:none;"
href="http://www.sfgate.com/cgi-bin/article.cgi?f=/c/a/2012/04/16/MNOE1O44CM.DTL">SFGate</a></span> The Associated Press won a Pulitzer Prize for investigative journalism Monday for revealing the New York Police Department's widespread spying on Muslims, while the <i>Patriot-News</i> in Harrisburg, Pa., and 24-year-old reporter Sara Ganim captured the award for local reporting for breaking the Penn State scandal that ultimately brought down Joe Paterno. <span style="font-weight:bold; text-decoration:none;"><a style="font-weight:bold; text-decoration:none;"
href="http://www.huffingtonpost.com/2012/04/16/huffington-post-pulitzer-prize-2012_n_1429169.html?1334603126">HuffPost</a></span> For eight months this past year, Huffington Post senior military correspondent David Wood reported extensively on the lives of severely wounded veterans and their families in "Beyond the Battlefield," a 10-part series awarded Monday with the Pulitzer Prize in National Reporting. <span style="font-weight:bold; text-decoration:none;"><a style="font-weight:bold; text-decoration:none;"
href="http://lens.blogs.nytimes.com/2012/04/16/pulitzer-prizes-the-effects-of-war-at-home/"><i>NYT</i> / Lens</a></span> Craig F. Walker of <i>The Denver Post</i> was awarded his second Pulitzer Prize on Monday afternoon for "Welcome Home: The Story of Scott Ostrom," which won in the feature category. News / The Cutline</a></span> The <i>Tuscaloosa News</i> won a Pulitzer for its coverage of 2011's deadly tornado -- including the paper's use of "social media as well as traditional reporting to provide real-time updates, help locate missing people and produce in-depth print accounts even after power disruption forced the paper to publish at another plant 50 miles away." <span style="font-weight:bold; text-decoration:none;"><a style="font-weight:bold; text-decoration:none;"
href="http://www.philly.com/philly/news/20120417_Inquirer_wins_Pulitzer_Prize_for_school_violence_series.html"><i>Philadelphia Inquirer</i></a></span> The <i>Inquirer's</i> investigation of a climate of pervasive violence in Philadelphia's public schools won the Pulitzer Prize for public service Monday. <span style="font-weight:bold; text-decoration:none;"><a style="font-weight:bold; text-decoration:none;"

function correctPNG() // correctly handle PNG transparency in Win IE 5.5 & 6.
{
   var arVersion = navigator.appVersion.split("MSIE")
   var version = parseFloat(arVersion[1])
   if ((version >= 5.5) && (document.body.filters)) 
   {
      for(var i=0; i<document.images.length; i++)
      {
         var img = document.images[i]
         var imgName = img.src.toUpperCase()
         if (imgName.substring(imgName.length-3, imgName.length) == "PNG")
         {
            var imgID = (img.id) ? "id='" + img.id + "' " : ""
            var imgClass = (img.className) ? "class='" + img.className + "' " : ""
            var imgTitle = (img.title) ? "title='" + img.title + "' " : "title='" + img.alt + "' "
            var imgStyle = "display:inline-block;" + img.style.cssText 
            if (img.align == "left") imgStyle = "float:left;" + imgStyle
            if (img.align == "right") imgStyle = "float:right;" + imgStyle
            if (img.parentElement.href) imgStyle = "cursor:hand;" + imgStyle
            var strNewHTML = "<span " + imgID + imgClass + imgTitle
            + " style=\"" + "width:" + img.width + "px; height:" + img.height + "px;" + imgStyle + ";"
            + "filter:progid:DXImageTransform.Microsoft.AlphaImageLoader"
            + "(src=\'" + img.src + "\', sizingMethod='image');\"></span>" 
            img.outerHTML = strNewHTML
            i = i-1
         }
      }
   }    
}
var browser = navigator.appName;
if (browser == "Microsoft Internet Explorer") {
   window.attachEvent("onload", correctPNG);
}
